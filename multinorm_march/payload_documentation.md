
# Документация параметра `payload`

## Назначение

`payload` — словарь параметров запроса к поисковой системе (retrieval API).  
Определяет текст запроса, режим поиска, уровни структуры документа и ограничения на количество результатов.

---

## Структура

```python
payload = {
    "text": "Какие требования к оперативной памяти?",
    "mode": "hybrid",
    "level": [0, 1, 2],
    "limit": 5,
    "use_summary": False
}
```

---

## Параметры

### `text: str`

Текст пользовательского запроса.

- Обязательный параметр
- Поддерживает естественный язык
- Используется как вход для поискового алгоритма

Пример:

```python
"text": "Какие требования к оперативной памяти?"
```

---

### `mode: str`

Режим поиска.

Допустимые значения:

- `"dense"` — семантический поиск (по embeddings)
- `"sparse"` — поиск по ключевым словам
- `"hybrid"` — комбинированный режим (dense + sparse)

Рекомендации:

- `dense` — для смысловых и абстрактных вопросов
- `sparse` — для точного совпадения терминов
- `hybrid` — универсальный режим (рекомендуется)

---

### `level: List[int] or int`

Уровни структуры документа, по которым выполняется поиск.

По умолчанию:

```python
[0, 1, 2]
```

Возможные значения:

- `0` — документ целиком (используется его summary)
- `1` — раздел документа
- `2` — подпункт / вложенный блок

Примеры:

Поиск только по разделам:

```python
"level": 1
```

Поиск по документу и разделам:

```python
"level": [0, 1]
```

---

### `limit: int`

Максимальное количество возвращаемых результатов (чанков).

- Тип: целое число
- Значение должно быть больше 0
- Рекомендуемое значение: 5–10

Пример:

```python
"limit": 10
```

---

### `use_summary: bool`

Использовать ли summary разделов (уровень 1) при поиске.

- Тип: логическое значение (`True` / `False`)
- По умолчанию: `False`

Рекомендации:

- Желательно включать при `mode="dense"` или `mode="hybrid"`
- Ускоряет поиск
- Повышает качество ответа для обобщённых запросов
- Может снижать точность для узкоспециализированных технических запросов

Пример:

```python
"use_summary": True
```

---

## Примеры конфигураций

### Универсальный режим

```python
{
    "mode": "hybrid",
    "level": [0, 1, 2],
    "limit": 5,
    "use_summary": True
}
```

### Поиск по точным терминам

```python
{
    "mode": "sparse",
    "level": [1,2],
    "limit": 10,
    "use_summary": False
}
```

### Семантический обзор документа

```python
{
    "mode": "dense",
    "level": [0, 1,2],
    "limit": 5,
    "use_summary": True
}
```
